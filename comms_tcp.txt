Acciones en `tcpdump`:

### 1. Mostrar interfaces de red disponibles

    tcpdump -D


### 2. Capturar un número específico de paquetes y guardarlos en un archivo

tcpdump -c <número_de_paquetes> -w <archivo.pcap>

Ejemplo para capturar 100 paquetes y guardarlos en `captura.pcap`:

tcpdump -c 100 -w captura.pcap


### 3. Seleccionar una interfaz específica para capturar paquetes

tcpdump -i <nombre_de_interfaz>

Ejemplo para capturar en `eth0`:

tcpdump -i eth0


### 4. Capturar paquetes específicos (TCP, UDP, ICMP, etc.)

tcpdump -i <interfaz> <protocolo>

Ejemplo para capturar solo paquetes TCP en `eth0`:

tcpdump -i eth0 tcp

Ejemplo para UDP:

tcpdump -i eth0 udp


### 5. Mostrar solo paquetes en ASCII

tcpdump -A


### 6. Capturar con fecha y hora

tcpdump -tttt


### 7. Guardar la captura en un archivo `.pcap` y mostrar el número de paquetes capturados

tcpdump -i <interfaz> -w <archivo.pcap>

Para contar los paquetes en el archivo:

tcpdump -r <archivo.pcap> | wc -l


### 8. Filtrar por dirección IP o rango de red (mostrando IP y puerto)
- Filtrar por IP de origen:

  tcpdump -i <interfaz> src host <IP>

- Filtrar por IP de destino:

  tcpdump -i <interfaz> dst host <IP>

- Filtrar por origen y destino:

  tcpdump -i <interfaz> host <IP>

- Filtrar por red:

  tcpdump -i <interfaz> net <rango_de_red>

- Mostrar IPs y puertos en formato numérico:

  tcpdump -n


### 9. Filtrar por puerto / tipo de puerto / rango de puertos
- Filtrar por puerto específico:

  tcpdump -i <interfaz> port <puerto>

- Filtrar por puerto de origen:

  tcpdump -i <interfaz> src port <puerto>

- Filtrar por puerto de destino:

  tcpdump -i <interfaz> dst port <puerto>

- Filtrar por rango de puertos:

  tcpdump -i <interfaz> portrange <puerto_inicio>-<puerto_fin>


### 10. Capturar paquetes con destino a una IP y puerto específico

tcpdump -i <interfaz> dst host <IP> and port <puerto>

Ejemplo para capturar tráfico con destino `192.168.1.1` en el puerto `23`:

tcpdump -i eth0 dst host 192.168.1.1 and port 23


### 11. Capturar paquetes con IP destino específica y puerto 80 o 443

tcpdump -i <interfaz> dst host <IP> and (port 80 or port 443)

Ejemplo:

tcpdump -i eth0 dst host 192.168.1.1 and (port 80 or port 443)


12. Capturar tráfico filtrando por tamaño del paquete

    tcpdump -i <interfaz> greater <tamaño_bytes>

Ejemplo para capturar paquetes mayores a 1000 bytes:

    tcpdump -i eth0 greater 1000

Para paquetes menores a un tamaño específico:

    tcpdump -i <interfaz> less <tamaño_bytes>


### Salir
Para detener la captura, usa `Ctrl + C`.
